# Region Time Counter Pro

**Расширенный счётчик длительности регионов для REAPER**

Region Time Counter Pro — это кроссплатформенный скрипт для REAPER,
который считает длительность регионов по принципу *union*
(без двойного учёта перекрытий).

Проект разработан в рамках **34dev.tools**.

---

## Возможности

- Total — суммарная длительность всех регионов (union)
- Selected — длительность отмеченных регионов (union)
- Список регионов с чекбоксами
- Поиск по имени региона
- Отображение длительности каждого региона
- Shift-клик — выделение диапазона
- Скролл колесом и перетаскиваемый скроллбар
- Навигация стрелками ↑ / ↓
- Сохранение выбора регионов в проекте
- Не требует js_ReaScriptAPI
- Работает на macOS и Windows

---

## Установка

1. Скачайте `Region Time Counter Pro.lua`
2. Поместите файл в папку Scripts REAPER
3. Откройте **Actions → Show action list**
4. Нажмите **ReaScript → Load**
5. Запустите скрипт
