# Region Time Counter Pro

**34dev.tools**

Профессиональный инструмент для REAPER, который считает длительность регионов **без двойного учёта перекрытий**  
и позволяет удобно работать с выбранными регионами через чекбоксы.

Разработано: **34dev — Audio Tools by Alexey Vorobyov (34birds)**

---

## Возможности

- **Total** — суммарная длительность всех регионов (union, без двойного учёта перекрытий)
- **Selected** — суммарная длительность отмеченных чекбоксами регионов (тоже union)
- Список регионов с чекбоксами и длительностью каждой строки
- **Поиск** по имени региона (и по `R<ID>`)
- Кнопки **All / Clear**
- **Shift-клик** — выделение диапазона
- **Колесо мыши + скроллбар** с перетаскиванием
- Клавиши **↑ / ↓** для прокрутки
- Состояние чекбоксов **сохраняется в проекте**
- Работает **без js_ReaScriptAPI** (чистый `gfx`)

---

## Требования

- REAPER 6+
- Не требует расширений или сторонних API

> Интерфейс написан на `gfx`, поэтому работает на macOS / Windows / Linux.  
> На Windows возможна небольшая разница в шрифтах (используется fallback).

## Установка

### Вручную (рекомендуется)

1. Скачайте файл  
   `ReaScripts/34birds_Region Time Counter Pro.lua`
2. В REAPER откройте  
   **Actions → Show action list… → ReaScript → Load…**
3. Выберите файл скрипта
4. Запустите его из Action List  
   (по желанию назначьте хоткей или добавьте в тулбар)

---

## Использование

1. Создайте регионы в проекте
2. Запустите **Region Time Counter Pro**
3. Отмечайте регионы чекбоксами:
   - клик — переключение
   - Shift-клик — диапазон
4. Используйте поиск для фильтрации списка
5. Значения **Total** и **Selected** обновляются автоматически

---

## Важно знать

- `Selected` считается **только по чекбоксам внутри скрипта**,  
  а не по выделению в Region/Marker Manager
- Перекрывающиеся регионы учитываются **один раз**
- Чекбоксы сохраняются в проекте и восстанавливаются при следующем запуске
